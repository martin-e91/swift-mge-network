Class Data​Task​Operation 

public final class DataTaskOperation<RequestType, DataType>: CompletionOperation<DataType, NetworkError>
where RequestType: Requestable, RequestType.ResponseType == DataType

An operation for network calls, using data tasks generated from the given session. Performs a NetworkRequest a completion upon its task termination.
